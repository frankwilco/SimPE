Diese Software befindet sich im Alpha Stadium!

SimPe kann Probleme in "Die Sims 2" oder am gesammten System verursachen. Wensie diese Software installieren akzeptieren sie dass die Entwickler in keiner Form für durch die Software verursachte Schäden verantwortlich sind.

SimPe verwendet die 7-Zip Kommandozeilen Version um Archive (wie z.B. .7z, .rar...) zu entpacken oder zu erstellen. 7-Zip wurde unter der GPL veröffentlicht. Nähere Informationen gibt es auf der Seite http://www.7-zip.org/.

Die neuen GUI Elemente wurden von divElements (http://www.divil.co.uk/net/) und SteepValley (http://www.steepvalley.net/) programmiert.

In der Standardeinstellung wird SimPe gelegentlich nach aktuelleren Versionen suchen. Dazu wird eine Internetverbindung aufgebaut, und unsere Infoseite aufgerufen. Es werden keine persönlichen Daten oder ähnliches übertragen oder gespeichert. Die Funktion kann in SimPE unter Extras->Einstellungen deaktiviert werden (Beim start auf Updates prüfen).
